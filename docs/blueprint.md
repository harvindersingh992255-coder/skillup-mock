# **App Name**: SkillUp Interview Ace

## Core Features:

- AI Interview Simulation: Simulate realistic job interviews using dynamic question generation based on job roles and industries powered by GPT-4o.
- Real-Time Speech Analysis: Analyze spoken responses using speech recognition, offering instant performance feedback.
- Body Language Detection: Tool that analyzes the user's body language during the interview and provide feedback on areas of improvement
- Multi-Role Scenario Customization: Offer customizable interview scenarios for different positions, industries, and difficulty levels.
- Progress Tracking Dashboard: Provide visual analytics showing improvement over time, skill development, and performance history in the Supabase PostgreSQL database.
- Personalized Learning Recommendations: Offer customized practice recommendations and curated content based on user performance and target roles, retrieved from the Supabase PostgreSQL database.
- Interview Recording & Playback: Allow audio recording for self-review, analysis, and sharing using Supabase Storage.

## Style Guidelines:

- Primary color: Deep blue (#1E40AF) for conveying professionalism and trust.
- Background color: Very light blue (#F0F5FF).
- Accent color: Emerald green (#10B981) for positive feedback and success states.
- Headings: 'Inter' (sans-serif) for a modern, clean, and highly legible look.
- Body Text: System UI fonts (sans-serif) for platform-optimized performance and readability.
- Use professional and clean icons representing various interview stages and feedback types.
- Subtle animations and transitions to provide feedback and guide the user through the interview process.
